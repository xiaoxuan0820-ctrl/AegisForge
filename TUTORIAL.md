还记得抽屉里那台屏幕碎了但还能开机的安卓机吗？那台三年前换下来的华为/小米/OPPO？

智能手机 = 触摸屏 + 扬声器 + 摄像头 + 麦克风 + WiFi + CPU。而一台 AI 助手的硬件需求也是这些。唯一的区别是 —— 它需要一个能操控屏幕、理解指令的大脑。

CiCi 就是这个大脑。

## CiCi 是什么

CiCi 是一款开源的 Android APK。装到手机上，开启无障碍服务，它就能：

1. **接收你的消息** —— 通过钉钉/飞书/QQ/Discord/Telegram 任一通道
2. **理解你的意图** —— LLM 大模型把自然语言翻译成操作指令
3. **操控手机** —— 通过无障碍服务点击、滑动、输入文字
4. **拍照回传** —— 每步操作后截图确认，让你远程看到结果

一句话：**你发消息，它操作手机，截图给你看结果。**

## 三步上手

### 第一步：装 App

从 [GitHub Releases](https://github.com/xiaoxuan0820-ctrl/AegisForge/actions) 下载最新 APK，安装到闲置安卓手机上。

> 要求：Android 8.0+，不需要 Root

### 第二步：开权限

1. 设置 → 辅助功能 → 已安装的服务 → 找到 CiCi → **开启**
2. 确认手机连着 WiFi，记下 IP 地址（设置 → 关于手机 → 状态信息 → IP 地址）

### 第三步：配通道

电脑浏览器打开 `http://你的手机IP:9527`：

```
1. LLM 配置：
   API Key:    sk-你的DeepSeek密钥
   Base URL:   https://api.deepseek.com
   模型名称:    deepseek-chat

2. 选一个消息通道（比如飞书）：
   App ID:     你的飞书应用ID
   App Secret: 你的飞书应用密钥

3. 保存
```

然后在飞书里给 Bot 发消息：「打开设置看看能做什么」，它就动了。

飞书里发：「帮我设置每天早晚打卡」。CiCi 会创建一个技能：

## 常见问题

**Q: 需要 Root 吗？**
不需要。基于 Android 无障碍服务 (AccessibilityService)，这是官方提供的辅助功能 API。

**Q: 支持什么手机？**
Android 8.0 (API 26) 以上的任何手机。华为/小米/OPPO/vivo/三星/一加都可以。

**Q: 用的是什么大模型？**
默认推荐 DeepSeek（便宜好用），也支持 OpenAI、Anthropic，以及任何兼容 OpenAI API 的模型服务。

**Q: 我发「打开淘宝」它就打开了吗？**
是的。它先截屏 → LLM 分析当前屏幕 → 移动到桌面 → 找到淘宝图标 → 点击 → 截屏确认 → 告诉你完成了。

**Q: 会不会把我的银行卡密码泄露？**
系统提示词明确禁止自动填写密码和支付信息。遇到登录/付费界面，它会停下来通知你，而不是自动操作。

**Q: 耗电吗？**
待机几乎不耗电（只有无障碍服务后台运行）。定时任务触发时，系统自动亮屏执行，完成后自动息屏。

**Q: 误触怎么办？**
3 次自动重试 + 自动回退机制。如果某一步操作后屏幕状态不符合预期，它会自动回退重试。同一步骤连续失败 3 次，自动 back 后重新规划。

## 进阶：自己开发技能

如果你会一点编程，可以在 `patches/hermes-agent/src/main/java/com/apk/claw/android/skill/SkillManager.kt` 的 `getBuiltinSkills()` 方法里添加你自己的内置技能：

```kotlin
Skill(
    id = "my_skill",
    name = "我的技能",
    description = "这是我自定义的技能",
    icon = "🛠",
    category = SkillManager.Category.CUSTOM,
    triggerType = SkillManager.TriggerType.DAILY,
    triggerTime = "9:00",
    steps = listOf(
        SkillManager.Step("打开XX应用"),
        SkillManager.Step("搜索YY"),
        SkillManager.Step("截图发送")
    )
)
```
